## 🎯 About

> A curated collection of **beautiful, high-performance web animations** built with modern front-end technologies.
> From smooth **CSS transitions** to immersive **3D WebGL experiences**, creative **UI/UX motion design**, and interactive **Next.js** apps — this is your go-to playground for the art of motion on the web. 🚀

```bash
git clone https://github.com/AryanSinghh07/Animations.git
cd Animations
npx serve .
```
